MPC-BEV Simulator

A Python-based control simulation combining Model Predictive Control (MPC) for vehicle path tracking with a simplified Battery-Electric Vehicle (BEV) model. It integrates steering and torque control to maintain precise trajectory tracking while modeling energy consumption, aerodynamic drag, and regenerative braking. The simulator visualizes real-time motion, steering effort, and battery performance, achieving smooth path following with over 97% SOC efficiency.

Renewable_Energy_Impact

This project explores how increasing renewable energy penetration affects wholesale electricity prices in Great Britain. Using real-world electricity market data and a mathematical model based on the merit-order effect, the analysis simulates how different levels of renewable integration (High, Moderate, Low) could shape future pricing outcomes through 2040. The project combines data cleaning, exploratory analysis, and scenario modeling in Python to demonstrate that greater renewable penetration can significantly reduce electricity costs. The findings are supported by historical trends and validated through international case studies (e.g., Germany and South Australia).

Nuga_bank_etl_using_pyspark

This project involved modernizing Nuga Bank's data exploration and cleaning processes using PySpark to address inefficiencies and scalability challenges. By automating data workflows, normalizing datasets into 2NF/3NF, and storing cleaned data in PostgreSQL, the solution improved data quality, scalability, and collaboration. Key outcomes included faster insights, reduced manual effort, and enhanced decision-making capabilities. Tools used: PySpark, Python, SQL, PostgreSQL, and GitHub.
